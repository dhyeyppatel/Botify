FAQ / Troubleshooting

❓ My bot is not responding

Ensure you clicked Start in the dashboard, and you’re messaging the correct bot username.

❓ Invalid token error

Double-check you copied the token fully from @BotFather. If leaked, revoke it and generate a new one.

❓ My command errors out

Go to the bot card → Errors. The last error will show which line failed.

❓ Can I run multiple bots?

Yes, up to 3 on Free plan. Upgrade to Pro for more.
